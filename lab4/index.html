Lab 4 - Pseudocoding and Problem-solving

Challenge

The challenge was to write pseudocode for a computer game that would later be turned into comments. We were also instructed to add our lists and indent as needed in the index.html as well as the lab.js to include our pseudocode turned into comments to make space for the java code.

Problems

Problems that arose were that I wasn't sure what to do the pseudocode on and how to do the command+/ to turn everything into a comment.

Results

 
						How to Clean a Bathroom
						Locate supplies
						If you have all the supplies, proceed to the Bathroom
						Set up supplies
						Clean objects starting from the objects highest in the room to objects bottom of the room
						If done cleaning, check cleanliness.
						If cleanliness does not pass, loop back to step 4.
						Return the supplies
						
						
						Rock Paper Scissors game
						Display valid options for the user to pick.
						Accept user input
						The computer chooses an option at “random”
						Test win condition or draw condition
						If user or computer won, print winner
						End Game
						If draw, loop back to first step
